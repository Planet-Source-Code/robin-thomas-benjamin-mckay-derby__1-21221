About Our Ring


 Formed in September 1997, The Microsoft Agent Ring is worldwide organization of websites featuring Microsoft Agent technologies.  Around our ring, you will find many of the best web sites, applications, and character designers in the world that are utilizing these fascinating technologies in the most innovative and coolest ways...

What is "Microsoft Agent"?   Microsoft® Agent is a set of programmable software services that supports the presentation of interactive animated characters within the Microsoft Windows® interface. Developers can use characters as interactive assistants to introduce, guide, entertain, or otherwise enhance their Web pages or applications.  Microsoft Agent enables software developers and Web authors to incorporate a new form of user interaction, known as conversational interfaces, that leverages natural aspects of human social communication. In addition to mouse and keyboard input, Microsoft Agent includes optional support for speech recognition so applications can respond to voice commands. Characters can respond using synthesized speech, recorded audio, or text in a cartoon word balloon.
What is a "Ring"?  A Ring is collection of web sites that have a similar interest linked together in a ring. Links on each web site in the ring (like those shown below) allow ring surfers (that's you!) to move from one web site to the next, eventually traversing the entire ring and ending up where they started.  There are thousands of rings in existence today covering nearly every topic imaginable.

Ring Info

December 3, 2000

THE MICROSOFT AGENT RING HAS MOVED TO RINGSURF.COM

Like thousands of other rings upset with the Yahoo! acquisition of WebRing.org and its redesign, stripped-down functionality, privacy issues, and a long list of other problems, The Microsoft Agent Ring has moved ring hosts to RingSurf.com.

If you are currently a ring member and have not updated the ring code on your site for RingSurf.com, please go to the "Join Our Ring" page and resubmit your site.

Ring Order
The Microsoft Agent Ring order is automatically rearranged weekly to help distribute traffic to all member sites, so clicking "Previous Site", "Next Site", etc. on a specific site's page will take you to a different site each week.

All comments, suggestions, and complaints regarding our ring are welcome. If you find a site that is offensive, missing the ring logo and links, or is no longer available - please let us know.
